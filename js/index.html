Где кто

Как узнать, что за значение у нас есть. Число это, или строка, или массив или еще что.

instanceof

Где кто. Number

Но:

typeof NaN === 'number'

Где кто. String, Boolean, Function

Где кто. null и undefined

x === undefined

При этом:

Где кто. RegExp

Где кто. Array

При этом x instanceof Array работает с проблемами.
